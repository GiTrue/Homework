# Advertisement Service (Asyncio + aiohttp)

Данный проект представляет собой REST API для сервиса объявлений, написанный с использованием асинхронного подхода.

## Технологии в проекте
* **Python 3.11**
* **aiohttp** — асинхронный веб-фреймворк.
* **SQLAlchemy 2.0 (Async)** — ORM для работы с базой данных.
* **PostgreSQL** — основное хранилище данных.
* **Pydantic** — валидация входящих данных.
* **Docker & Docker Compose** — контейнеризация приложения.

---

## Задание 1: Асинхронное API

Реализованы асинхронные роуты для управления объявлениями:
- `POST /advertisements` — создание объявления (заголовок, описание, владелец).
- `GET /advertisements/{id}` — получение информации об объявлении.
- `PATCH /advertisements/{id}` — редактирование.
- `DELETE /advertisements/{id}` — удаление.

**Особенности реализации:**
- Используется асинхронный драйвер `asyncpg`.
- Инициализация базы данных (создание таблиц) происходит автоматически при старте приложения.
- Валидация полей реализована через схемы Pydantic.

2. Склонируйте репозиторий и перейдите в папку проекта.

   ```bash
   docker-compose up --build
