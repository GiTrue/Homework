# Дипломный проект: Сервис автоматизации закупок

## Описание
Backend-приложение для автоматизации процесса закупок в ритейле, реализованное на Django REST Framework. 
Реализован импорт товаров, управление корзиной, статусы заказов и отправка уведомлений.

## Технологии
- Python 3.13 / Django 4.2
- Django REST Framework
- SQLite (локально) / PostgreSQL (рекомендуется для prod)
- Celery + Redis (очереди задач)

## Как запустить
1. Клонировать репозиторий.
2. Создать виртуальное окружение: `python -m venv venv`.
3. Установить зависимости: `pip install -r requirements.txt`.
4. Выполнить миграции: `python manage.py migrate`.
5. Запустить сервер: `python manage.py runserver`.

## Тестирование
Для запуска тестов выполните:
`python manage.py test`
