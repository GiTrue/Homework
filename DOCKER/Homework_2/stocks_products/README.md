# Склады и товары (Docker edition)

## Описание
REST API для управления запасами на складах. В данной версии проект полностью настроен для работы в **Docker** с использованием базы данных **SQLite3**.

---

## Как запустить через Docker

### 1. Сборка образа
Откройте терминал (Git Bash) в папке проекта и выполните команду:
```bash
docker build -t stocks-api .

2. Запуск контейнера
Запустите контейнер, пробросив порт 8000:

Bash
docker run -p 8000:8000 stocks-api

3. Проверка

Приложение доступно по адресу: http://localhost:8000/api/v1/

Типовые команды для работы

Применить миграции:

Bash
docker exec -it <ID_КОНТЕЙНЕРА> python manage.py migrate

Создать суперпользователя:

Bash
docker exec -it <ID_КОНТЕЙНЕРА> python manage.py createsuperuser
Особенности конфигурации для ДЗ
База данных: SQLite3 (перенастроено согласно заданию).

Секретный ключ: Передается через переменные окружения в Dockerfile.
