# Задание 1: Кастомный образ Nginx

Данный проект демонстрирует создание собственного Docker-образа на базе Nginx с заменой стандартной страницы приветствия.

## Как запустить

1. **Сборка образа:**
   ```bash
   docker build -t my-nginx-image .

2. Запуск контейнера:

Bash

docker run -d -p 8080:80 --name nginx-container my-nginx-image

3. Проверка: Откройте браузер по адресу http://localhost:8080. 

Вы увидите измененную страницу приветствия.
