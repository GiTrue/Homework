## Image Upscale Service (Flask + Celery + OpenCV)
Этот сервис предоставляет API для увеличения разрешения (апскейлинга) изображений с использованием модели искусственного интеллекта EDSR (Enhanced Deep Residual Networks).

## Особенности реализации

- Эффективность: Модель загружается в память воркера один раз и переиспользуется для всех последующих задач.

- Работа в памяти: Изображения не сохраняются на диск сервера; обработка происходит в оперативной памяти через байтовые буферы.

- Асинхронность: Обработка тяжелых изображений происходит в фоновом режиме через Celery и Redis.

## Инструкция по запуску

1. Подготовка модели
Поместите файл модели EDSR_x2.pb в директорию models/ внутри корня проекта.

2. Сборка и запуск
Убедитесь, что у вас установлены Docker и Docker Compose. Выполните команду в терминале:

```bash

docker-compose up --build

```bash


После завершения сборки:

API будет доступно по адресу: http://localhost:5000

Redis будет запущен на порту 6379


## Использование API

Шаг 1: Отправка изображения на обработку

Отправьте POST запрос с файлом (ключ file).

    ```bash

curl -X POST -F "file=@path_to_your_image.png" http://localhost:5000/upscale
    
	```bash
	

Шаг 2: Проверка статуса

Используйте полученный task_id, чтобы узнать состояние обработки.

    ```bash

curl http://localhost:5000/tasks/<task_id>
    
	```bash
	
Ответ (в процессе): {"status": "PENDING"}
Ответ (готово): {"status": "SUCCESS", "link": "/processed/<task_id>"}


Шаг 3: Скачивание результата

Перейдите по ссылке из ответа или используйте:

    ```bash

curl http://localhost:5000/processed/<task_id> --output upscaled_image.png
    
	```bash
	

## Технологический стек

- Python 3.11

- Flask — веб-интерфейс.

- Celery — очередь задач.

- Redis — брокер сообщений и хранилище результатов.

- OpenCV (dnn_superres) — движок для работы с ИИ-моделью.
