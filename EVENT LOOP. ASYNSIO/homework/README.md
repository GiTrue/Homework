Как запустить проект:

1. Скопируйте все файлы в одну папку.

2. Откройте терминал в этой папке.

3. Запустите команду:

Bash

docker-compose up --build

4. Скрипт app дождется готовности базы данных, создаст таблицы и начнет выводить прогресс. По завершении в базе данных swapi появятся все персонажи.
